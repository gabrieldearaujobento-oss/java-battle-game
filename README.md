# Jogo de Batalha Java

Sistema de batalha em Java utilizando Programação Orientada a Objetos.

## Funcionalidades

* Criação de personagens
* Sistema de ataque
* Controle de vida
* Verificação de personagem vivo/morto

## Tecnologias utilizadas

* Java
* Programação Orientada a Objetos (POO)

## Estrutura do projeto

* src/

## Objetivo

Projeto desenvolvido para prática de lógica de programação, encapsulamento e conceitos de POO.
